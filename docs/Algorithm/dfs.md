# Traversal 순회

| 순회방법은 root를 언제 방문하냐에 따라 세가지로 나눈다

- 재귀라는 개념이 중요

* Preorder

  - root -> left -> right

* Inorder

  - left -> root -> right

* Postorder

  - left -> right -> root
