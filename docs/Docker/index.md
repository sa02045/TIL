### Hypervisor(가상화) vs Docker

- Hypervisor의 역할

  - 가상화 기술은 VM이 운영체제의 도움없이 직접적으로 하드웨어에 접근할 수 있도록 도와줌 + 최적화

- Docker
  - Docker에 의해 App이 격리된 환경에서 돌아감
  - 별도의 OS도 없고 VM도 없음
  - 거의 성능이 99퍼 그대로 나옴

### Docker의 기반 기술

- Namespaces

  - 자원을 공유하고 있는 프로세스으 ㅣ그룹

- Cgroups
- chroot
